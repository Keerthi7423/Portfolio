  Full Stack Developer & Mobile Focus
  Revision: 1.0 | Date: April 2026

PROJECT DESCRIPTION:
A high-end, cinematic web portfolio inspired by Iron Man/Stark Industries UI.
Features a 3D Hero scene, scroll-driven animations, classified dossier sections,
and a JARVIS AI Agent that interacts via voice and text.

CORE CONSTRAINTS:
- No overlapping dependencies (Sequential workflow)
- Each task is atomic (One thing at a time)
- Estimated Duration: ~28 Business Days (Approx. 5.5 Weeks)

================================================================================
PHASE 1: ENVIRONMENT & INFRASTRUCTURE (Days 1-3)
================================================================================

Task 1.1: Project Initialization  (0.5 Days)
- Initialize Next.js 14 project (App Router, JS, Tailwind).
- Setup .gitignore and initial folder structure as per Tech Stack PRD.

Task 1.2: Dependency Installation (0.5 Days)
- Install Three.js, R3F, Drei, GSAP, Framer Motion, Lenis, and Resend.
- Verify package.json versions match approved tech stack.

Task 1.3: Design Tokens Implementation (1.0 Day)
- Standardize tailwind.config.js with --color-void, --color-marvel-red, etc.
- Configure Typography system: Bebas Neue, Rajdhani, Orbitron, IBM Plex.

Task 1.4: Global CSS & Utility Classes (1.0 Day)
- Implement app/globals.css with CSS variables for all design tokens.
- Add utility classes for scanlines, noise, and holographic glow effects.
